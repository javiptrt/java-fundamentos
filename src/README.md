# Calculadora Java

Calculadora básica desarrollada en Java que implementa las cuatro operaciones aritméticas fundamentales.

## Operaciones disponibles
- Suma
- Resta
- Multiplicación
- División

## Tecnologías
- Java 21
- IntelliJ IDEA

## Ejecución
Abre el proyecto en IntelliJ IDEA y ejecuta la clase `Calculadora`.
